006

Web scraping

ASL Fingerspelling is a sign language that uses hand gestures to represent letters and numerals. This project aims to recognize ASL hand gestures using deep learning neural networks, with the goal of improving communication between deaf individuals. The system architecture involves recording videos of ASL gestures, extracting frames, identifying body keypoints using PoseNet, and passing cropped frames to a CNN model for training. A segmentation algorithm is used to separate individual letters from words and combine them to identify words. The accuracy of both alphabets and videos can be predicted.
